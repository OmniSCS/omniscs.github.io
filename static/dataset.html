Dataset Information

Photorealism Dataset

We selected 8 challenging scenes from the public NuScenes dataset for scenario editing. These included scenes with over 50 vehicles, data from different countries, and varied weather (sunny, rainy) and lighting conditions (daytime, nighttime). Every tenth frame of the driving videos was used as a test frame.

Data Fidelity for Algorithms

Beyond photorealism, synthetic data for autonomous driving systems must also exhibit algorithmic realism. Following the real-world corner case CODA dataset setup, we selected 8 safety-critical scenarios for evaluation.
